FileSpec(...args: FS.PathSegment[])

Class representing a file system entry when it is known to be a file.

  • add(...args: string[]): FileSpec

    Appends additional path segments to the current file system path.

  • backup(opts?: Util.FileConflictStrategy): Promise<FileSpec | undefined>

    Backs up the current file by moving it to a new path, determined by the chosen strategy. This is useful for preserving a file before overwriting it. The moveTo operation is called with overwrite: true, ensuring the backup file is created.

  • basename(): FS.FileBasename

    Returns the file's base file name, minus it's extension. For example, for '/path/to/file.name.html', this method will return 'file.name'. Unlike node:path basename method, this does NOT include the extension.
